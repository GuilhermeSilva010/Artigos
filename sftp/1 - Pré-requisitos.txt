1 - Pré-requisitos
    1.1 - Configuração de variáveis de Ambiente
        A variável de ambiente JAVA_HOME deve estar configurada
        A variável de ambiente WEBLOGIC_HOME deve estar configurada para o diretório C:\[diretório do weblogic]\wlserver_10.3

1.2 - Instalando o Maven

    Baixe a versão 3.5.4 do Maven na página da Apache http://maven.apache.org/
    Descompacte o arquivo na unidade C:\ do computador
    Configure a variável de ambiente M2_HOME para: C:\apache-maven-3.5.4
    Edite a variável de ambiente Path adicionando ao final: ;%M2_HOME%\bin
    No prompt de comando, digite mvn -version e verifique se a versão que aparece é 3.5.4. Caso contrário, siga os passos novamente
    Ainda no prompt, digite mvn archetype:generate para criação da pasta .m2 no diretório C:\Users\[seu usuario]\.m2 e ignore o erro BUILD ERROR
    Copie o arquivo Settings.xml para a pasta .m2 no seu computador
    Novamente no prompt, digite mvn archetype:generate para criação da pasta repository-maven.mapa-nexus no diretório C:\Users\[seu usuario]\.m2\repository-maven.mapa-nexus
    Feche a janela do prompt
    Configure a variável de ambiente M2_REPO para: C:\Users\[seu usuario]\.m2\repository-maven.mapa-nexus

1.3 - Configurando o Eclipse

    Acesse o menu Window -> preferences
    Selecione Java -> Code Style
    Selecione o item Clean Up', clique em import e selecione o arquivo CleanUp.xml
    Clique em apply
    Selecione o item Code Templates', clique em import e selecione o arquivo CodeTemplates.xml
    Clique em apply
    Selecione o item Formatter', clique em import e selecione o arquivo Formatter.xml
    Clique em apply
    Selecione Java -> Build Path -> Classpath Variables
    Clique em New
    Name = M2_REPO
    Path = C:\Users\{SEU_USUARIO}\.m2\repository-maven.mapa-nexus\
    Clique em Ok
    Clique em Ok novamente

2 - Configurando o primeiro projeto
    2.1 - Abrindo um projeto no Eclipse
        Efetue o checkout do código-fonte do projeto, disponível no SVN, para o ambiente local (estação de trabalho)
        Clique em File -> import
        Selecione Maven -> Existing Maven Projects e selecione o diretório raiz do projeto
